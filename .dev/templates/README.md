# Шаблони для плагінів та тем

Ця папка містить заготовки (шаблони) для створення нових плагінів та тем для Flowaxy CMS.

## Структура

- `plugin-template/` - Шаблон для створення нового плагіна
- `theme-template/` - Шаблон для створення нової теми

## Швидкий старт

### Створення нового плагіна

1. Скопіюйте папку `plugin-template` в `plugins/`
2. Перейменуйте папку на назву вашого плагіна (наприклад, `my-plugin`)
3. Перейменуйте `PluginName.php` на назву вашого класу (наприклад, `MyPlugin.php`)
4. Оновіть `plugin.json` з інформацією про ваш плагін
5. Оновіть клас у файлі плагіна (змініть `PluginName` на назву вашого класу)
6. Детальні інструкції в `plugin-template/README.md`

### Створення нової теми

1. Скопіюйте папку `theme-template` в `themes/`
2. Перейменуйте папку на назву вашої теми (наприклад, `my-theme`)
3. Оновіть `theme.json` з інформацією про вашу тему
4. Оновіть `index.php` з вашим HTML/CSS/JS
5. Детальні інструкції в `theme-template/README.md`

## Примітки

- Всі шаблони містять базову структуру та приклади коду
- Ви можете видалити непотрібні файли та папки
- Додайте свої файли за потреби
- Перевірте документацію в README.md кожного шаблону
