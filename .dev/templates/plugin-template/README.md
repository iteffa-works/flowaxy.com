# Plugin Name

Опис плагіна

## Встановлення

1. Скопіюйте папку `plugin-template` в папку `plugins`
2. Перейменуйте папку на назву вашого плагіна (наприклад, `my-plugin`)
3. Перейменуйте файл `PluginName.php` на назву вашого плагіна (наприклад, `MyPlugin.php`)
4. Оновіть `plugin.json` з інформацією про ваш плагін
5. Оновіть клас у файлі плагіна:
   - Змініть `PluginName` на назву вашого класу
   - Оновіть namespace та опис

## Структура

```
plugin-name/
├── plugin.json          # Метадані плагіна (обов'язково)
├── PluginName.php       # Основний файл плагіна (обов'язково)
├── admin/               # Адмін-панель (опціонально)
│   ├── AdminPage.php    # Клас адмін-сторінки
│   └── template.php     # Шаблон адмін-сторінки
├── src/                 # Вихідний код (опціонально)
│   └── classes/
├── db/                  # SQL файли для встановлення (опціонально)
│   └── install.sql
├── assets/              # CSS, JS, зображення (опціонально)
│   ├── css/
│   └── js/
└── README.md            # Документація
```

## Використання

Після встановлення плагін буде доступний в адмін-панелі в розділі "Плагіни".
