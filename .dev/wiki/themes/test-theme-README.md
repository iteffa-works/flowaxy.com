# Test Theme

Тестовая тема для демонстрации работы системы хуков Flowaxy CMS.

## Возможности

Эта тема демонстрирует:

1. **Использование хуков в шаблонах**
   - `applyFilter()` - для модификации данных
   - `doAction()` - для выполнения действий
   - `hookManager()` - для получения статистики

2. **Регистрация хуков в functions.php**
   - Фильтры для модификации контента
   - События для вывода меню
   - Модификация заголовка

3. **Интеграция с плагинами**
   - Тема работает совместно с плагинами через хуки
   - Плагины могут модифицировать контент темы

## Установка

1. Тема уже находится в директории `themes/test-theme/`
2. Перейдите в админ-панель: `/admin/themes`
3. Найдите тему "Test Theme"
4. Нажмите "Активировать"

## Структура темы

```
test-theme/
├── theme.json          # Метаданные темы
├── index.php          # Основной шаблон
├── style.css          # Стили темы
├── functions.php      # Функции темы (хуки)
└── README.md          # Документация
```

## Хуки, используемые темой

### В index.php:
- `applyFilter('theme_title', ...)` - модификация заголовка
- `applyFilter('theme_content', ...)` - модификация контента
- `applyFilter('test_priority', ...)` - демонстрация приоритетов
- `doAction('theme_head')` - вывод в head
- `doAction('theme_footer')` - вывод в footer
- `doAction('theme_menu')` - вывод меню

### В functions.php:
- `addAction('theme_menu', ...)` - регистрация меню
- `addFilter('theme_title', ...)` - модификация заголовка
- `addFilter('theme_content', ...)` - модификация контента

## Совместная работа с плагинами

Тема разработана для работы совместно с плагином **Test Hooks Plugin**:

1. Активируйте тему
2. Активируйте плагин Test Hooks Plugin
3. Откройте главную страницу сайта
4. Вы увидите результат совместной работы темы и плагина через хуки

## Особенности

- Адаптивный дизайн (Bootstrap 5)
- Демонстрация статистики хуков
- Список всех зарегистрированных хуков
