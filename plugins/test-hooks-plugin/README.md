# Test Hooks Plugin

Тестовый плагин для демонстрации работы системы хуков и событий Flowaxy CMS.

## Возможности

Этот плагин демонстрирует:

1. **Фильтры (Filters)**
   - Добавление пункта меню в админку (`admin_menu`)
   - Добавление виджета на главную панель (`dashboard_widgets`)
   - Модификация контента темы (`theme_content`)
   - Демонстрация приоритетов (`test_priority`)

2. **События (Actions)**
   - Добавление стилей в head темы (`theme_head`)
   - Добавление скриптов в footer темы (`theme_footer`)
   - Логирование активации/деактивации плагинов

3. **Условное выполнение**
   - Фильтр `theme_content` выполняется только если контент не пустой

4. **Приоритеты**
   - Демонстрация работы приоритетов (5 и 20)

## Установка

1. Плагин уже находится в директории `plugins/test-hooks-plugin/`
2. Перейдите в админ-панель: `/admin/plugins`
3. Найдите плагин "Test Hooks Plugin"
4. Нажмите "Активировать"

## Использование

После активации плагин автоматически:

- Добавит пункт меню "Test Hooks" в админку
- Добавит виджет на главную панель с статистикой хуков
- Добавит стили и скрипты в тему
- Начнет модифицировать контент темы

## Хуки, используемые плагином

- `admin_menu` - фильтр для добавления пунктов меню
- `dashboard_widgets` - фильтр для виджетов
- `theme_head` - событие для вывода в head
- `theme_footer` - событие для вывода в footer
- `theme_content` - фильтр для модификации контента
- `plugin_activated` - событие активации плагина
- `plugin_deactivated` - событие деактивации плагина
- `test_priority` - демонстрация приоритетов

## Код
